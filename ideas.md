# rust-unification

## features

* Unification
* Coercion
* Matching

## unification expressions

* variable, ex: a
* symbol, example "FOO"
* kind (term): Foo
* list: [a, b, c]
* dictionary: {foo=a, bar=b}
* set: "FOO" | "BAR"
